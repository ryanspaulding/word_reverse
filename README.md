# Word Reverse 
Takes a file of sentences and the reverses the words. Empty lines from the input file will be ignored. 

## Example Input File

Hello World  
Hello Giraffe

Output will look like:

World Hello  
Giraffe Hello

### On the command line

word_reverse.rb example_sentences_to_reverse.txt
